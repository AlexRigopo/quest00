# Prompt Piscine – Quest 00

This repository contains a series of Markdown exercises designed to build foundational skills in reflection, reasoning, evaluation, and prompt‑craft within the Prompt Piscine program.

## Contents

| File | Description |
|------|-------------|
| `ex00_reflection.md` | Personal reflection on learning approach and mindset. |
| `ex01_palindrome.md` | Working through logic, pseudocode, and implementation of a palindrome checker. |
| `ex02_variations.md` | Generating structured variations of prompts or ideas. |
| `ex03_fairness_contract.md` | Drafting a fairness / responsibility contract for how you use AI. |
| `ex04_scenarios.md` | Analysis of hypothetical or real scenarios involving AI. |
| `ex05_irreplaceable_skills.md` | Identifying human‑unique skills and how to strengthen them. |

## How to Use This Pack

1. **Read each exercise** in order.  
2. **Write your own answers** directly in the corresponding `.md` files.  
3. **Iterate** as needed—many exercises are designed to be refined over time.  
4. **Use AI sparingly and intentionally**—the goal is to improve your independent reasoning.  

## Structure

All exercises are plain Markdown and require no dependencies.  
You can open them with any text editor or Git platform.
